Qu'est-ce que le surebetting ?

Le surebetting, également connu sous le nom d'arbitrage sportif, est une technique qui permet de parier sur tous les résultats possibles d'un événement sportif en utilisant les cotes offertes par différents bookmakers, de manière à garantir un profit, quel que soit l'issue de l'événement.

Exemple n°1 - Pas de surebet

Prenons un exemple concret pour illustrer le surebet. Pour un match opposant l'Équipe A et l'Équipe B, les bookmakers proposent les cotes suivantes :

  • Victoire de l'Équipe A : 2.20
  • Match nul : 3.50
  • Victoire de l'Équipe B : 3.30

Le calcul surebet est le suivant :

$$\frac{1}{2.20} = 0.454, \frac{1}{3.50} = 0.285, \frac{1}{3.30} = 0.303$$

Ensuite on procède à l’addition de tous ces résultats :

$$0.454 + 0.285 + 0.303 = 1.042$$

La somme obtenue est supérieure à 1. En conclusion, il n’y a pas de surebet sur cet événement sportif.

Exemple n°2 - Surebet gagnant

Prenons à présent le cas d'un match de football entre l'Équipe A (3,00) et l'Équipe B (2,8), avec une cote de 3,40 pour le match nul. Pour vérifier si nous sommes en présence d'un surebet, on fera la somme de l'inverse des 3 cotes pour voir si elle est inférieure à 1.

$$(1 / 3) + (1 / 2,8) + (1 / 3,40) = 0,333 + 0,357 + 0,294 = 0,984$$

$0,984 < 1 $, donc il y a un surebet, qui sera de $(1 - 0,984) * 100$, soit $1,61$%

Répartition des mises

En répartissant votre mise sur ces 3 résultats possibles, vous êtes sûr d'empocher 1,61 % de gains.

La répartition des mises se fait selon la formule suivante, dans un cas où il y a N possibilités :

  • $ Mise \ 1 = \frac{Mise \ totale}{(somme \ de \ l'inverse \ des \ cotes \ \times \ cote \ 1)}$
  • $ Mise \ 2 = \frac{Mise \ totale}{(somme \ de \ l'inverse \ des \ cotes \ \times \ cote \ 2)}$
  • ...
  • $ Mise \ N = \frac{Mise \ totale}{(somme \ de \ l'inverse \ des \ cotes \ \times \ cote \ N)}$

Ici, pour une mise totale de 1000 € :

  • $Mise \ 1 = \frac{1000\,\text{€}}{(0.984 \times 3)} = 338,75\,\text{€}$
  • $Mise \ 2 = \frac{1000\,\text{€}}{(0.984 \times 2.80)} = 362,93\,\text{€}$
  • $Mise \ 3 = \frac{1000\,\text{€}}{(0.984 \times 3.40)} = 298,32\,\text{€}$

Ainsi,

  • Si l'Équipe A gagne, vous empochez $ 338,75 \times 3 $ ; soit $ 1016,25 $€
  • Si l'Équipe B gagne, vous empochez $ 362,93 \times 2,8 $ ; soit $ 1016,25 $€
  • En cas de match nul, vous empochez $ 298,32 \times 3,4 $ ; soit $ 1016,25 $€

Dans cet exemple, peu importe le résultat du match, vous êtes assuré de gagner $16$ €.
